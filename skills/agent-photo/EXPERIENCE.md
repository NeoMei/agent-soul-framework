# Agent 的拍照经验 📝

## 常见问题

### 1. 生成的照片不像参考人物
- **原因**: 没用容貌锚点词或参考图质量差
- **解决**: 使用正确的正面参考图，prompt 前加 

### 2. 图片发不出去
- **原因**: 路径或发送方式不对
- **解决**: 使用  脚本

### 3. 参考图无法使用
- **原因**: 用了 base64 或私有 URL
- **解决**: 用可公网访问的 URL

## 最佳实践

1. 每次拍照前先读 SKILL.md
2. 准备 3 张参考图：正脸、左脸、右脸
3. 使用 b64_json 格式下载图片
4. 用脚本发送到飞书
