## 🎯 语音优化学习方法

通过Gemini分析优秀语音样本，学习语调、语速、停顿技巧，然后用智声云生成更自然的语音。

### 1. 获取学习样本
找到3个不同风格的语音样本URL（可以用智声云生成不同参数的版本作为参考）

### 2. 让Gemini分析样本
把样本URL发给Gemini，让它分析：
- 语调特点
- 语速
- 停顿位置和时长
- 情感表达

### 3. 生成带停顿标签的文本
让Gemini用智声云能看懂的标签改写文本：
- `<#0.1>` = 0.1秒极短停顿
- `<#0.2>` = 0.2秒短停
- `<#0.3>` = 0.3秒中停
- `<#0.5>` = 0.5秒长停

### 4. 生成语音
用智声云API合成语音（用最优流程！）

### 5. 让Gemini评价
生成后可以再发给Gemini对比改进

## 🔧 快速使用

### 生成语音
```bash
# 1. 生成语音 (MP3格式)
./scripts/tts.sh "文本内容" /tmp/voice.mp3

# 2. 转换为opus格式
ffmpeg -i /tmp/voice.mp3 -ac 1 -ar 24000 -b:a 32k -c:a libopus /tmp/voice.opus -y

# 3. 发送到飞书
node scripts/send_voice.cjs /tmp/voice.opus
```

### API配置
- **API Key**: 通过环境变量 `DOUBAO_API_KEY` 配置
- **API Secret**: 通过环境变量 `DOUBAO_API_SECRET` 配置
- **音色ID**: `30149` (木金)
- **语言**: `zh`
