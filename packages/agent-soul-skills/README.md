# @neomei/agent-soul-skills

> 魂器可选技能插件包 — 包含需要 Python 脚本的 Agent 技能

## 安装

```bash
npm install -g @neomei/agent-soul-skills
```

## 用途

本包为 `@neomei/agent-soul-framework` 的可选扩展，提供：

- 丰富的 Agent 技能（`skills/`）
- 历史 Python 脚本与工具（`scripts/`）
- 旧版心跳实现（`heartbeat/`）
- 连接器辅助脚本（`connectors/`）

安装后，运行 `hunqi init` 会自动将技能复制到项目目录的 `skills/` 下。

## 依赖

- Python 3
- `@neomei/agent-soul-framework`
